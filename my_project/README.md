# My Project

## Опис

Цей проєкт містить простий додаток на Python зі структурованими каталогами, тестами та базовими налаштуваннями lint.

## Команди Makefile

- `make lint` — перевірка коду flake8 та black.
- `make test` — запуск тестів за допомогою pytest.
